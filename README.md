# 欢迎进入 Web 3D 学习世界. ThreeJs

# 准备工作
```
非前端人员，需要安装nodejs 以及yarn。要掌握JS 简单语法 if, switch, var, const, let, new class(),function 函数 了解也可以学
```
[Nodejs 下载](https://nodejs.org/en/)
[Yarn 下载](https://yarn.bootcss.com/)
## 项目工程依赖包下载
```
yarn install
```

### 开始启用工程
```
yarn serve
```

### 工程编译
```
yarn build
```

### 工程编译
```
拉去dev 的分支.
```
